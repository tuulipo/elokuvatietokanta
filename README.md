# Elokuvatietokanta
 Yksinkertainen tietokantasovellus, jonka avulla käyttäjä voi pitää kirjaa katsomistaan ja häntä kiinnostavista elokuvista. 

* Sovelluksessa käyttäjät pystyvät lisäämään ja muokkaamaan elokuvia. Elokuvalle tulee antaa ainakin perustiedot (nimi, julkaisuvuosi, ohjaaja, kesto). Elokuvalle voidaan antaa muita tietoja (esim. synopsis, pääosan esittäjä).
* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
* Käyttäjä pystyy selaamaan tietokantaan lisättyjä elokuvia.
* Käyttäjä pystyy etsimään hakusanalla elokuvia.
* Käyttäjä pystyy arvostelemaan elokuvia (arvostelu = arvosana + halutessaan myös kommentti). Elokuvan tiedoissa näytetään kommentit ja keskimääräinen arvosana.
* Käyttäjä pystyy lisäämään elokuvan suosikiksi.
* Käyttäjäsivu näyttää kuinka monta elokuvaa käyttäjä on arvostellut ja listan käyttäjän tekemistä arvosteluista.
* Käyttäjä pystyy valitsemaan elokuvalle yhden tai useamman luokittelun (esim. genre, Oscar-voittaja).
* Käyttäjä pystyy tekemään muokkauspyynnön elokuvalle (esim. tietojen korjaus tai lisäys).

Pääasiallinen tietokohde on elokuva ja toissijainen tietokohde on elokuvan arvostelu.
